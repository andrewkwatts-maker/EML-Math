EML-Math

All of mathematics from a single operator · High-performance Rust backend · C/C++/Rust compatible

eml(x, y) = exp(x) − ln(y)

One binary operator reconstructs all 36 elementary functions — +, −, ×, /, exp, ln, sin, cos, π, e — and extends to Lorentz boosts, spacetime metrics, octonions, and Clifford algebras.

The Operator

The EML Sheffer operator is the continuous analog of NAND for Boolean logic. Odrzywolek (2026) proved that eml(x, y) = exp(x) − ln(y) together with the constant 1 generates all of elementary mathematics:

from eml_math import EMLPoint
import math

EMLPoint(1, 1).eml()           # e  = 2.71828...
EMLPoint(2, 1).eml()           # exp(2) = 7.38905...

# Nesting EMLPoints builds any expression
EMLPoint(1, EMLPoint(EMLPoint(1, math.e), 1)).eml()  # ln(e) = 1.0

What's New in v0.5.0

Lorentz Geometry

EMLPoint.boost(φ), minkowski_delta(), rapidity(), light_cone_type() — relativistic mechanics built from the EML coordinate pair.

Spacetime Metrics

MetricTensor with 8 built-in geometries: flat, Schwarzschild, FLRW, Calabi–Yau, Klebanov–Strassler, Heterotic E₈×E₈, AdS₅×S⁵, G₂.

Octonions & Lattices

Octonion with Fano-plane product. E₈ lattice (240 roots). Leech lattice. EMLNDVector for N-dimensional EML spaces.

Clifford Algebra

EMLMultivector — full geometric product ab = a·b + a∧b, rotors, grade projection. Unifies all metric constructions in one operator.

Rust Batch Functions

boost_n, octonion_mul_n, geometric_product_n, christoffel_batch_n — Rayon-parallel batch operations, up to 9× faster than Python.

C API

Standalone libeml_math (static + shared) with no Python dependency. Use from C, C++, or as a Rust crate. Includes eml_math.h header.
