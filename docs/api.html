API Reference

All public symbols are importable from eml_math directly. Submodule imports are also supported for fine-grained access.

EMLPoint

from eml_math import EMLPoint

Method / PropertyDescription
EMLPoint(x, y, D=None)Create a node. x, y can be floats or nested EMLPoints. D enables discrete mode.
.tension() → floatexp(x) − ln(y) — the EML Sheffer operator. Alias: .eml()
.mirror_pulse() → EMLPointOne Mirror-Pulse step: (x, y) → (|y|, T). Alias: .pulse()
.frame_shift() → EMLPointForced Axiom-8 frame shift using |y|.
.is_slipping() → boolTrue if x > OVERFLOW_THRESHOLD (Slipping Wheel).
.conserves_tension(nxt, tol) → boolVerify Axiom 10 identity.
.resonates_with(other, tol) → boolAxiom 14: tension ratios match.
.is_leaf() → boolTrue if both coords are plain floats (Rust delegation enabled).
.diff(var) → EMLPointSymbolic differentiation w.r.t. a named variable.

Geometry (v0.5.0)

MethodDescription
.pair() → EMLPair(exp(x), ln(|y|)) as an EMLPair.
.euclidean_delta() → float√(exp(2x) + (ln y)²) — Euclidean frame invariant. Rust-accelerated.
.minkowski_delta(signature="+---", c=1.0) → float√|exp(2x) − (c·ln y)²|. Rust-accelerated.
.is_timelike(c) → boolexp(2x) > (c·ln y)²
.is_spacelike(c) → boolexp(2x) < (c·ln y)²
.is_lightlike(c, tol) → bool|exp(2x) − (c·ln y)²| < tol
.light_cone_type(c) → str"timelike", "spacelike", or "lightlike".
.light_cone_coordinates(c) → (float, float)Null coordinates (t+x, t−x).
.rapidity() → floatatanh(ln(y)/exp(x)). Raises ValueError for spacelike.
.boost(phi, c=1.0) → EMLPointLorentz boost by rapidity φ. Δ_M invariant. Rust-accelerated.
.boost_velocity(v, c=1.0) → EMLPointBoost by velocity v. Raises if |v| ≥ c.
.canonical_frame(k=0) → EMLPairRotate pair into frame k ∈ {0,1,2,3} (×1, ×i, ×−1, ×−i).
.rest_energy(c) → floatAlias for minkowski_delta().
.proper_time(c) → floatrest_energy() / c.
